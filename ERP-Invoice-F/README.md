# BillStack Business Manager

BillStack Business Manager is a comprehensive Enterprise Resource Planning (ERP) and Customer Relationship Management (CRM) application. Built with a modern tech stack, it provides robust solutions for accounting, invoicing, quoting, and inventory management tailored for modern businesses.

## 🚀 Tech Stack

### Frontend
*   **React.js**: Core UI library.
*   **Ant Design (AntD)**: Comprehensive UI component library for enterprise-grade design.
*   **Redux Toolkit**: State management for handling complex application states (CRUD operations, authentication, advanced settings).
*   **Vite**: Fast, modern frontend build tool.
*   **React Router**: For seamless Single Page Application (SPA) navigation.

### Backend (API integration)
*   **Node.js**: The backend environment powering the API.
*   **RESTful API Integration**: Securely communicates with a custom backend utilizing JWT for authentication and secure data transmission.

## 🌟 Key Features & Modules

### 1. Inventory & Product Management
*   **Comprehensive Product Tracking**: Manage products with details like Name, category, Price, Quantity, description, and HSN Codes.
*   **Automated SKU Generation**: SKUs are automatically and securely generated by the backend to ensure data integrity, and remain safely locked from frontend manipulation.
*   **Dynamic Categories**: Asynchronous associative data selection, intelligently mapping product groups.

### 2. Quotations & Invoicing
*   **Quotes**: Generate, manage, and track estimates and quotes for clients.
*   **Invoicing**: Seamlessly convert quotes to invoices, track payment status, and manage complex billing cycles.
*   **Terms & Conditions**: Manage default and custom terms applied to invoices and quotes, complete with internal logic for default selection.

### 3. System & Company Configurations
*   **Company Settings**: Centrally manage the organization's profile, including customized tax numbers, registration IDs, and localized data.
*   **General Settings**: Deeply customize application behavior, spanning from specific date formatting to default app configurations.

### 4. Advanced User Interface
*   **Dynamic Forms**: A highly reusable form engine (`DynamicForm/index.jsx`) that handles multiple input types, asynchronous lookups, complex validation rules, and dynamically disabled constraints.
*   **Resilient Data Tables**: Intelligent data grids designed to flawlessly handle asynchronous populated objects and dynamic pagination from varied API responses, ensuring zero application crashes.
*   **Responsive Design**: A fluid layout ensuring usability across desktops, tablets, and varying screen sizes.

## 🛠️ Internal Architecture Highlights

Over the course of its development, BillStack Business Manager has been engineered for extreme resilience:
*   **Safeguarded Redux State**: Redux actions meticulously parse highly-variable API payload structures to guarantee reliable `items` and `pagination` mapping. 
*   **Object-to-String React Parsing**: Sophisticated mapping algorithms safely dissect complex nested MongoDB objects (like populated `Category` records) directly into raw React text, outright eliminating standard "Object is not a valid React child" errors.
*   **Form Security Filtering**: Advanced frontend constraints restrict the user's ability to arbitrarily mutate critical read-only fields (like internal IDs and SKUs) while maintaining their absolute visibility during auditing.

## 📦 Getting Started

### Prerequisites
*   Node.js (v16+)
*   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone [your-repository-url]
    cd [your-project-directory]
    ```

2.  **Install Frontend Dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Variables**:
    Ensure your `.env` configuration points to the correct backend endpoint:
    ```env
    VITE_BACKEND_SERVER=http://localhost:8888/api
    ```

4.  **Start the Development Server**:
    ```bash
    npm run dev
    ```
    The application will be accessible at `http://localhost:3000`.
